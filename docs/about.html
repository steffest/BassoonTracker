About

BassoonTracker is a free online music creation app following the long tradition of so-called "tracker" music programs.

It runs in your browser and produces real-time generated music, using samples, notes, and effects.

BassoonTracker handles the popular .mod and .xm file formats.

Most notable features are:

  • 100% plain old JavaScript - no dependencies!
  • Open Source (MIT License)
  • Protracker and FastTracker compatible: load, edit and save .mod and .xm files
  • Allows up to 32 channels
  • Has a built-in sample editor
  • Midi support
  • Imports 8-bit WAV files, 8-Bit Sampled Voice (8SVX) files, and RAW samples (as well as any other format the AudioContext interface can read, such as .mp3 and .ogg)
  • Exports to .wav and .mp3 formats
  • Includes the historic ST-01 and ST-02 sample disks, released in 1987 (as well as a bunch of others)
  • Uses the modarchive API to give access to several thousands of music files
  • Uses the modules.pl API to give access to even more music files
  • Can connect to your Dropbox account to directly read and write your own files
  • Includes a lightweight player library which you can use in your own projects

If words like Trackers, Samples, MOD and XM don't sound familiar and you're interested in the roots of home-produced digital music, read the "What is a tracker" section, or watch this highly entertaining video to get a feel of what's it all about.
