Playlists

BassoonTracker supports playlists. Some nice ones are included in the "Playlists" menu.
When playings songs from a playlist, the next song will automatically start when the current one ends.
You can control the playlist with the buttons at the bottom, as well as active shuffle play.
You can save and load playlists from the file menu.
The default playlist format is .pls that you can also use with other tracker-supporting software like VLC or XMPlay.
M3u and json formats are also supported.
External playlists can be load from a URL using the "file" parameter. (example)

A particular nice way to explore is the "Random" playlist. This will generate a list of 20 random songs from the Modarchive database, that currently holds over 50.000 MOD and XM songs.


If more curated lists are your cup of tea, you can check out the playlists that are collected by the nice folks at hippoplayer.se that are nicely integrated in BassoonTracker.

Favorites

a special type of playlist is your favorites list. You can add/remove songs to it by clicking the heart icon on the right;
Your favorites list is stored in your browser's local storage.
If you want to save it to a file, you can save it as a playlist.
